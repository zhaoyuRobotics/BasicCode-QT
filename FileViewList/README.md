# documentList  

在 ViewList 控件中以图标加文字的形式显示某一目录下的文件  

文件所在目录详细说明：

（1）.cpp 程序文件所在目录 viewList

（2）.jpg的图片放在目录 resourse

（3） .txt的文件存放在目录 config
